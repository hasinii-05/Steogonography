# Steganography Web Application

This project is a frontend-only web application built using React and TypeScript.  
It demonstrates basic client-side steganography concepts by allowing users to hide and extract secret messages within images using browser-based image processing.

## Features
- Client-side image steganography demonstration
- Encode secret text into an image
- Decode hidden text from an image
- No backend or database required
- Runs entirely in the browser

## Technologies Used
- React
- TypeScript
- Vite
